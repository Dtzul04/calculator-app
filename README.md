# Calculator App

A simple and responsive calculator app built using **HTML**, **CSS**, **JavaScript**, and **Bootstrap**. This app allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- User-friendly interface with **Bootstrap** for responsive design.
- Performs basic arithmetic operations (addition, subtraction, multiplication, and division).
- Clear button to reset the calculator.
- Responsive design that works well on mobile and desktop devices.
